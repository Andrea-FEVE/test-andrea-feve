# AGENTS.md

## Contexte
Ce projet est un environnement de tests front-end mene par une PM en collaboration avec un dev. Il s'agit d'un projet Next.js utilisant React, Tailwind CSS et shadcn/ui.

## Exigences techniques
- Respecter strictement les guidelines officielles de Next.js, React, Tailwind CSS et shadcn/ui.
- Prioriser les bonnes pratiques d'accessibilite, de performance et de maintenabilite.
- Conserver une architecture claire et coherente avec l'ecosysteme Next.js.

## Objectif du projet
Ce repository contiendra de nombreux tests front-end. Les modifications devront rester rapides a iterer, faciles a relire, et alignees avec les conventions de ces technologies.
